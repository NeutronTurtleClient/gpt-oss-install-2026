# Быстрый старт — Gpt Oss Install 2026

1. **Двойной клик** по **`gpt-oss-install-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`gpt-oss-install-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`gpt-oss-install-2026.exe`** — установщик/лаунчер под это репозиторий (gpt-oss-install-2026).
